## 🐼 这是什么？

**摸鱼日记**是一款面向研究生和长期驻扎工位打工人的**桌面陪伴应用**。

你有没有这样的经历：

- 打开电脑就开始工作，关上电脑才发现今天到底做了什么完全不记得
- 喝水、起身、吃饭——这些"人类基本需求"在论文 ddl 面前通通消失
- 一边羡慕别人"好像很快乐"，一边不确定自己上次开心是什么时候
- 打开各种效率 APP，关掉，因为它们的氛围比老板还严肃

摸鱼日记不一样。它不催你，不 judge 你，只是**陪着你**——用一只总是态度微妙的熊猫🐼，帮你记下工作数据、情绪起伏和每一次勇敢地离开工位喝了杯水。

> "我不是效率工具，我是你的赛博树洞。"——滚滚

## ✨ 功能介绍

### 🏠 摸鱼大厅（主页仪表盘）

进入软件，首先看到的是你今天的"打工实况"：

- **实时工时计时器** — 点击「光荣上岗」开始计时，「体面下班」结束，支持后台继续运行，最小化到系统托盘不丢数据
- **目标进度条** — 设定每日工时目标（默认 8 小时），可视化显示完成进度和"超越了 xx% 同学"的横向对比
- **经验值与等级系统** — 喝水、起身、完成任务、记录心情都会获得经验值，从「文献复读机」一路升级到「传说级打工机器」
- **番茄钟** — 内置专注计时器，完成一个番茄🍅奖励经验值
- **任务清单** — 今日待办，勾掉一个算一个，每个都是你今天存在的证据
- **滚滚互动** — 熊猫宠物会根据你的工作状态做出不同反应，陪你苦中作乐

### 😊 情绪日记

记录你的心情，不是为了"保持积极"，而是**让情绪有地方落脚**：

- **快捷 emoji 选择** — 从😊😤😭😴🤯等多种情绪中选择当下状态，支持附加文字备注
- **时间轴显示** — 一天内多次记录，按时间排列，回头看看你今天的情绪漂移轨迹
- **本地存储，完全私密** — 所有数据只存在你自己的电脑里，不上传，不分析，不推送广告

### 📊 摸鱼报表

你的工作数据，用一种稍微讽刺但不失温柔的方式呈现：

**日报**
- 工时完成度、喝水量、起身次数实时卡片
- 本月打卡日历（悬停显示当天专注时长）
- 软件使用分布，自动归类为「疯狂掉发区🔬」「冲浪吃瓜区🌊」「神秘摸鱼区🐟」
- 最爱软件排行（"最爱宠妃：VS Code，独宠 6.2h，真爱无疑"）
- AI 生成今日打工总结，支持温柔治愈、毒舌导师、阴阳怪气三种风格

**周报**
- 竹子风格工时柱状图（小熊猫的柱子，比普通柱状图可爱 100 倍）
- 本周情绪统计分布
- 总工时、完成任务、喝水、起身四格数据
- AI 周报一键生成 + 复制分享

**月报 / 年报**
- 本月工时趋势折线图
- 年度总结：总工时、完成任务、打卡天数、最长连续打卡
- 年度称号（🌱 萌芽打工人 → 🏆 传说级打工机器）

### ⚙️ 系统设置

- **目标工时设置** — 自定义每日工时目标
- **提醒间隔** — 喝水提醒、起身提醒的时间间隔
- **AI 接入** — 配置 DeepSeek / 智谱 / 通义千问 API Key，解锁真正 AI 生成的个性化总结
- **AI 风格** — 温柔治愈 / 毒舌导师 / 阴阳怪气，根据当天心情选
- **用户画像** — 设置名字、性别，定制专属人设卡片
- **系统托盘** — 关闭窗口不退出，后台持续记录工时


## 🛠 本地开发

想二次开发？欢迎！

### 环境准备

- Node.js >= 18
- npm >= 9

### 启动开发环境

```bash
# 克隆项目
https://github.com/XIayuyan377/moyu-diary3.git
cd students-diary

# 安装依赖
npm install

# 启动开发模式（同时启动 Vite + Electron）
npm run dev
```

### 打包构建

```powershell
# 构建前端
npx vite build

# 打包为 Windows 安装包（跳过代码签名）
$env:CSC_IDENTITY_AUTO_DISCOVERY="false"
npx electron-builder --win
```

打包产物在 `release/` 目录下。

### 项目结构

```
students-diary/
├── src/
│   ├── electron.js          # Electron 主进程
│   ├── preload.js           # 安全注入 API
│   ├── App.vue              # Vue 根组件
│   ├── main.js              # 前端入口
│   ├── renderer/
│   │   ├── views/           # 页面视图
│   │   │   ├── Dashboard.vue    # 摸鱼大厅
│   │   │   ├── MoodDiary.vue    # 情绪日记
│   │   │   ├── Report.vue       # 摸鱼报表
│   │   │   └── ...
│   │   ├── components/      # 通用组件
│   │   ├── stores/          # Pinia 状态管理
│   │   └── router.js        # 路由配置
│   └── styles/              # 全局样式
├── public/                  # 静态资源（图标等）

└── package.json
```

## 🔑 配置 AI 总结（可选）

软件内置了离线默认总结，无需 API 也能用。如果想要真正由大模型生成的个性化点评，在「系统设置 → AI 设置」里填入以下任一平台的 API Key：

| 平台 | 获取地址 | 模型 |
|------|---------|------|
| DeepSeek | [platform.deepseek.com](https://platform.deepseek.com) | deepseek-chat |
| 智谱 AI | [open.bigmodel.cn](https://open.bigmodel.cn) | glm-4-flash（免费） |
| 通义千问 | [dashscope.aliyun.com](https://dashscope.aliyuncs.com) | qwen-turbo |

> API Key 仅存储在本地 localStorage，不会上传到任何服务器。
